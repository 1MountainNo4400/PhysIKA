32 /*
33  * ArrayND<ElementType,Dim>: Dim-dimensional array with elements of ElementType
34  * If Dim==1, it's 1D array defined array.h
35  * We separate 1D array in Array class because we want to keep the simple
36  * form of Array<ElementType> for 1D array instead of Array<ElementType,1>
37  * Hence:
38  * Use Array<ElementType> for 1D array and ArrayND<ElmentType,Dim> for higer dimension
39  * ArrayND<ElementType,1> will result in compiler error due to the static assert
40  *
41  * Dim is arbitrary, but most probably it's 2 and 3.
42  * Elements of ArrayND can be accessed via index or iterator, the index is represented as
43  * a std::vector. Physika::Vector can also be used as index if and only if Dim = 2,3,4.
44  */
45
